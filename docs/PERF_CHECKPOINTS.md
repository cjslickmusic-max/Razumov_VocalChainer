# Performance checkpoints

Журнал **сравнимых** замеров CPU (и при необходимости RAM) плагина. Заполнять вручную после релиза или при крупных изменениях DSP/графа.

## Как измерять (ориентир)

- Один и тот же **хост**, **буфер**, **частота**, **цепочка** (`chainProfile`), **пресет** / макросы на нейтрали.
- Смотреть загрузку CPU в хосте или инструменте OS; для A/B — **одна** сессия, два билда подряд.
- Фиксировать **версию** (`CMake` `PROJECT_VERSION` / строка в UI).
